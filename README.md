# C++ Technical Interview Preparation 🚀

This repository contains my solutions to various algorithmic challenges and data structure implementations, specifically curated for **C++ technical interview** preparation.

The goal is to demonstrate proficiency in **Modern C++**, efficient problem-solving, and clean, documented code.

## 🛠 Tech Stack
- **Language:** C++17 / C++20
- **Compiler:** `g++` (GCC) / `clang`
- **Focus:** STL (Containers & Algorithms), Memory Management (RAII), and OOP Principles.

## 🌿 Branching Strategy
To keep the main history clean and organized, exercises are separated by **branches**. Each branch focuses on a specific category or problem set:
- **`main`**: Overview, roadmap, and core utility templates.
- **`topic/data-structures`**: Implementations of Trees, Graphs, and Linked Lists.
- **`topic/algorithms`**: Sorting, Searching, and Dynamic Programming.
- **`feat/leetcode-solutions`**: Specific problem sets from LeetCode.
